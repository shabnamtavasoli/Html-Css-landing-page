# HTML & CSS Landing Page

A single-page website built with HTML5 and CSS3, featuring a navigation menu and multiple sections.  
This project is created as a sample portfolio project.

## Features
- Internal navigation menu
- Multiple distinct sections with IDs
- Built using HTML5 and CSS3
- Includes images and custom fonts

## Technologies
- HTML5
- CSS3
